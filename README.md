# 🎮 Minecraft 2D (Browser Game)

> A browser-based 2D sandbox game inspired by Minecraft and Terraria.  
> **Current version: alpha 0.0.5** — now with block breaking, placing, and an inventory!

🔗 **Play now:** [minecraft2d-alpha.netlify.app](https://minecraft2d-alpha.netlify.app)

---

## ✨ Features

- ✅ **Procedural world generation** – hills, grass blocks, dirt layers.
- 🧱 **Collisions & gravity** – stand on blocks, fall if there's no ground.
- 🏃 **Auto-jump system** – automatically steps up one-block hills.
- ⛏️ **Destroy & place blocks** – Left-click destroys OR places a block, depending on what's selected in the inventory.
- 🎒 **Basic inventory** – 10 slots, choose a block to place.
- 🖱️ **Grid-based placement** – every block is 50×50 pixels.
- 🎮 **Keyboard movement** – Arrow keys (Left, Right, Up, Down).
- 🔧 **Built with pure HTML/CSS/JS** – no frameworks, runs in any browser.

---

## 🎯 Controls

| Key            | Action                     |
|----------------|----------------------------|
| ← / →          | Move left/right (auto-jump)|
| ↑              | Jump (manual)              |
| ↓              | Fall down / step off       |
| 🖱️ Left click  | Destroy OR place block (depends on inventory selection) |
