# 🎮 Minecraft 2D (Browser Game)

> A browser-based 2D sandbox game inspired by Minecraft and Terraria.  
> **Current version: alpha 0.0.4** — collisions, falling, auto-jump, and world generation with hills & grass. you can test it on minecraft2d-alpha.netlify.app

---

## ✨ Features (so far)

- ✅ **Procedural world generation** – hills, grass blocks, dirt layers.
- 🧱 **Collisions & gravity** – you stand on blocks and fall if there's no ground.
- 🏃 **Auto-jump system** – automatically steps up one-block hills.
- 🖱️ **Grid-based block placement** – every block is 50×50 pixels.
- 🎮 **Keyboard movement** – Arrow keys (Left, Right, Up, Down).
- 🔧 **Built with pure HTML/CSS/JS** – no frameworks, runs in any browser.

---

## 🎯 Controls

| Key          | Action        |
|--------------|---------------|
| ← / →        | Move left/right (with auto-jump) |
| ↑            | Jump (manual override) |
| ↓            | Fall down / step off |

*(In alpha 0.0.5 you'll be able to destroy and place blocks!)*
